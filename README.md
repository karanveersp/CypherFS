# CypherFS

![cypherfs logo](./images/steel_logo.PNG)

A CLI password manager.

Use it to encrypt text you want to keep, or share securely.

![cypher fs demo](./images/cypherfs_demo.gif)

## Installation

Download the `.exe` from the _Releases_ section.
